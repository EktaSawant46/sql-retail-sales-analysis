# Retail Sales Analysis using SQL (Beginner Level)

## Project Overview
This beginner-level SQL project focuses on analyzing a retail sales dataset using
SQL queries executed in **SQL Server Management Studio (SSMS)**.

The project demonstrates how SQL can be used to create tables, explore data,
and answer basic business-related questions from retail sales data.

## Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)

## Dataset Description
The dataset represents retail sales transactions and includes information such as:
- Transaction date and time
- Customer details
- Product category
- Quantity sold
- Price per unit
- Total sales amount

The dataset is provided in CSV format and was used to populate the SQL table.


## Objectives
- Create and manage a retail sales database using SQL
- Perform basic data exploration
- Analyze sales data to answer simple business questions
- Practice writing SQL queries in SSMS

## Key Analysis Performed
- Created database and table structure
- Inserted retail sales data into the table
- Counted total sales transactions
- Identified unique customers and product categories
- Filtered sales data using WHERE conditions
- Analyzed sales based on dates and categories
- Answered basic business-oriented queries using SQL

## Conclusion
This project helped strengthen my understanding of SQL fundamentals such as
database creation, table management, data filtering, aggregation, and
writing structured queries using SQL Server Management Studio.
